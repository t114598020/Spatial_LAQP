### This is spatial database final project.
Install PostgreSQL(18.1, Windows) at [Postgre](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads?referrer=grok.com).

Install [POWER dataset](https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption).

Install [uber pickups NYC](https://drive.google.com/file/d/1pdWmns1IamjDkMToYapufxCmtb8kFDp3/view?usp=sharing).

### Environment

Python 3.10.19

Run `pip install -r requirements.txt` for the environment.

### To test postgis
Windows search "pgAdmin 4" 

Right click to the postgres and select Query Tool.

Input `SELECT PostGIS_Version();` at the code section and execute.

If the Data Output shows version then done.
